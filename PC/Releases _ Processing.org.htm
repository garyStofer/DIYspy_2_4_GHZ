Stable Releases

Read about changes between Processing 3.0 and Processing 4.0.
The list of revisions covers the differences between releases in detail.

  • Version 4.3 -(July 26, 2023)
  • Version 3.5.4 -(January 17, 2020)
  • Version 2.2.1 -(July 31, 2014)

Note that these legacy releases are provided for archival purposes. We cannot provide support for them. If you need to update your code to work with the last version, please consult the changes page. If you encounter any issues with the current release, please submit a bug report. You can also build older releases from the source code. Note: Android Mode requires Processing 3 or later.
