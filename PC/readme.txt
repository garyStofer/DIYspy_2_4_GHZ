This folder contains the PC side display program for tghe DIYspy 2.4ghz scanner.

It is in the form of a processing script (.pde) and requires to be loaded and run under the 
processing IDE.  

The zipfile contains an installation of the Processing IDE V3.5.4 to be installed on the target PC 
before the the program "diyspy_fast.pde" can be loaded and run.
